## Why aliases?

Type less, do more. Aliases turn repetitive commands into short keystrokes so you stay in flow.

## The file

```bash title="~/.zsh/aliases.zsh"
## Navigation
alias ..='cd ..'
alias ...='cd ../..'
alias ~='cd ~'
alias c='clear'

## Lists
alias ll='ls -lah'
alias la='ls -A'
alias l='ls -lh'

## Git
alias g='git'
alias gs='git status -sb'
alias ga='git add'
alias gaa='git add .'

alias gcm='git commit -m'
alias gcmsg='git commit -m'
alias gp='git push'
alias gl='git pull'
alias gb='git branch'
alias gco='git checkout'
alias gd='git diff'
alias gds='git diff --staged'

## Package Manager
alias prb='pnpm run build'

## Save My Ass
```

## Modern CLI replacements

The Modern CLI Replacements section is commented out by default. Uncomment the ones you want after installing the tools:

| Alias  | Replaces | Tool                                                      |
| ------ | -------- | --------------------------------------------------------- |
| `ls`   | ls       | [eza](https://github.com/eza-community/eza)               |
| `cat`  | cat      | [bat](https://github.com/sharkdp/bat)                     |
| `grep` | grep     | [ripgrep (rg)](https://github.com/BurntSushi/ripgrep)     |
| `find` | find     | [fd](https://github.com/sharkdp/fd)                       |
| `du`   | du       | [dust](https://github.com/bootandy/dust)                  |
| `top`  | top      | [btop](https://github.com/aristocratos/btop)              |
| `diff` | diff     | [delta](https://github.com/dandavison/delta)              |

Install all of them at once with Homebrew:

```bash
brew install eza bat ripgrep fd dust btop git-delta
```

## How to install

1. Create the file:

```bash
mkdir -p ~/.zsh && touch ~/.zsh/aliases.zsh
```

2. Paste the aliases above into `~/.zsh/aliases.zsh`.

3. Source it from your `~/.zshrc` by adding this line:

```bash title="~/.zshrc"
source ~/.zsh/aliases.zsh
```

4. Reload your shell:

```bash
source ~/.zshrc
```
